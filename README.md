# UAS-CLOUD
# Cloud Computing Implementation Using Vercel

Implementasi website statis sederhana sebagai media pembelajaran Cloud Computing menggunakan layanan Vercel. Project ini dibuat untuk memenuhi tugas Ujian Akhir Semester mata kuliah Cloud Computing Program Studi Teknik Informatika Universitas Muhammadiyah Sukabumi.

---

## Overview

Website ini merupakan implementasi sederhana dari layanan Cloud Computing dengan memanfaatkan platform Vercel sebagai media deployment. Seluruh website dibangun menggunakan satu file `index.html` yang berisi struktur HTML, CSS, dan JavaScript sehingga proses deployment menjadi lebih sederhana dan mudah dipelajari.

---

## Objectives

Project ini dibuat dengan tujuan sebagai berikut.

- Memahami konsep dasar Cloud Computing.
- Menerapkan deployment website menggunakan layanan cloud.
- Menggunakan GitHub sebagai Version Control System.
- Mengintegrasikan GitHub dengan Vercel.
- Memahami proses Continuous Deployment.
- Menghasilkan website yang dapat diakses secara online.

## Deployment Workflow

```
Local Development
        │
        ▼
 GitHub Repository
        │
        ▼
 Automatic Deployment
        │
        ▼
      Vercel
        │
        ▼
 Live Website
```

Setiap perubahan pada branch `main` akan otomatis dideteksi oleh Vercel dan langsung dilakukan proses build serta deployment tanpa konfigurasi tambahan.

---

## Running Locally

1. Clone repository.

```
git clone https://github.com/username/UAS-CLOUD.git
```

2. Masuk ke folder project.

```
cd UAS-CLOUD
```

3. Buka file `index.html` menggunakan browser.

Tidak diperlukan proses instalasi maupun build.

---

## Deployment to Vercel

1. Login ke akun Vercel.
2. Import repository GitHub.
3. Pilih repository `UAS-CLOUD`.
4. Klik Deploy.
5. Tunggu proses deployment selesai.
6. Website dapat langsung diakses melalui URL yang diberikan oleh Vercel.
